A very cool quick view popup plugin for WooCommerce. Let your customers see extra product details and place and order without leaving the current page!

== Description ==

Pretty simple. This plugin let's your customers take a quick look (aka a quick view) at extra product data and even place an order by opening a popup in the current page. The popup can contain the product details, the image(s) and the options for placing an order. Works with all product types!

**Plugin Features**

* Compatible with most themes, including: StoreFront, Avada, Flatsome, Divi + more.
* Plug & play: just install it and it's ready to use!
* Modern glass effect
* Hide/Show on
    * Mobile
    * Tablet
    * Desktop
* Choose the position of the quick view button
* Choose the colors and text of the button
* Compatible with: Variation Swatches for WooCommerce
* Edit visually using the customizer
